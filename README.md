## 🧠 Prompt Framework: HBC-X

All prompts follow the **HBC-X Framework** — a professional UGC ad engineering structure:

```
H — Hook      → First 3 seconds — scroll-stopping pattern interrupt
B — Body      → Problem → Agitation → Solution → Proof storytelling
C — CTA       → Conversion action with urgency trigger + friction reducer
X — eXecution → Platform format rules, tone calibration, forbidden phrases
```

## 💡 Key Learnings

1. **Hooks are the entire ad** — 95% of ad success is determined in the first 3 seconds
2. **Persona specificity unlocks domain knowledge** — "Indian D2C UGC specialist" produced culturally relevant output
3. **Platform differences are real** — same product needs completely different energy per platform
4. **Forbidden phrases > style instructions** — telling AI what NOT to say is more effective
5. **Hinglish is a strategy** — culturally calibrated language produces higher authenticity for Indian audiences
